# DataAnalysis

A collection of exploratory data analysis projects and notebooks.

This repository is intended to document my growth in statistics, programming, and applied data analysis as a Statistics and Economics student.

## Project purpose

The goal of this repository is to practice the full data analysis workflow:

1. Ask a clear question
2. Load and clean data
3. Explore patterns
4. Visualize results
5. Build simple models
6. Communicate conclusions clearly

## Skills demonstrated

- Python programming
- Data cleaning
- Exploratory data analysis
- Visualization
- Statistical reasoning
- Communication of results

## Recommended project format

Each analysis should include:

- Research question
- Dataset source
- Cleaning steps
- Exploratory visualizations
- Statistical or machine learning method
- Results
- Limitations
- Next steps
